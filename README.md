# personal-website

All the code for my personal website: caydenjohnson.com.

The site functions as a place where I can host personal 
information about me and all my social media links including:

* Github
* LinkedIn
* X
* Instagram
* YouTube

I also plan on adding a blog later where I can discuss tech and
other random topics that interest me.

I've created this site for a variety of reasons. First, I like 
the idea of owning a small part of the internet that belongs 
soley to me. There' a great deal of independence 
that comes with owning your own site and being in charge of its
content and security that you don't get with just social media 
accounts. Secondly, this site serves as a portal to access all 
of the content I produce and acts as a 
centralized way to see what projects I'm working on. Finally, 
creating this site gives me ample opportunity to practice full 
stack web development skills and allows me to apply all the 
concepts that I've learned along the way. 

As a side note, all code for this website is AI free. I write 
all my own code, relying on documentation and stack overflow, 
just like the good old days before 2022. I use AI all the time 
and have no opposition to it whatsoever, but  as this site also
serves as a practice tool, it beats the point of learning web 
development if I just outsource all the coding to a large language model.
